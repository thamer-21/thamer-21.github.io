📦 Installation

  1. Open your project in Unity 2021.3 LTS or newer.
  2. Go to Window → Package Manager → My Assets.
  3. Find the RealCut asset you purchased and click Download, then Import.
  4. Keep all files selected and press Import. A RealCut/ folder appears in your project.

No extra packages or scripting-define symbols are required — every asset is pure C# with zero dependencies.

🚀 Quick Start

Smart Save System

// 1. Add the SmartSaveManager prefab to your first scene.
// 2. Mark fields you want persisted:
public class PlayerStats : MonoBehaviour
{
    [SmartSave] public int coins;
    [SmartSave] public float health;
}

// 3. Save or load anywhere:
SmartSave.Save();          // write current state
SmartSave.Load();          // restore last save
SmartSave.Save("slot-2");  // named save slots

Advanced Inventory System

// 1. Create items: right-click → Create → RealCut → Item Definition
// 2. Drop the InventoryPanel prefab into your Canvas.
// 3. Add items from code:
Inventory.Main.Add(itemDefinition, amount: 3);

// Listen for changes:
InventoryEvents.OnItemAdded += item =>
    Debug.Log($"Picked up {item.DisplayName}");

⚙️ Settings Reference

SettingDefaultDescription

Save Interval120 sSeconds between auto-saves. Set to 0 to disable timed saves.

💡 Examples

  • RPG Inventory — grid inventory with equipment slots and tooltips.
  • Crafting Bench — recipe setup, queues and result callbacks.
